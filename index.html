Project Description

We are currently developing a full stack web application that students can interact with to learn about robotics principles. Our goal is to bring more attention and opportunity for students to learn about the importance of robotics as there is a lack of interest and availability in terms of education and resources to learn about the field of robotics. WARE will serve as a central hub consisting of multiple learning exercises where users can immerse themselves with robotics principles and concepts and hone their knowledge of robotics while completing the various goals and exercises in each environment. In each environment, users can write or upload their own Python code and instantly see the visual output on the website with the use of an open source library from OpenAI called Gym.

Users and Instructors will be able to create their own personal accounts. Students will be able to use a ‘Class ID’ code when creating their accounts to be grouped with their instructor. This then enables an instructor to choose which environments the students will be able to have access to, as well as being able to observe, monitor, and evaluate their students’ progress in the various environments. The application is written with performance and simplicity in mind, as all processing of the user’s code is done on the backend server. Consequently, users using low power devices such as smartphones and tablets can take advantage of this learning platform without sacrificing performance. This application is being designed to be available on all devices.
